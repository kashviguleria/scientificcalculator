# ScientificCalculator
This project is a Scientific Calculator web application built using Streamlit and Python. It provides both basic arithmetic operations and advanced mathematical functions in a simple and interactive user interface.

 Features
Basic operations: Addition, Subtraction, Multiplication, Division
Scientific functions:
Trigonometric functions (sin, cos, tan)
Logarithmic calculations
Square root and power functions
Constants like π (pi) and e
Interactive UI with buttons and display
Real-time calculation output
Error handling for invalid inputs
